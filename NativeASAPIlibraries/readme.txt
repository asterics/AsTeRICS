
This folder contains the Native ASAPI Libraries for interfacing devices with C code:

- GSM Modem library (GSMModemLibrary)
- Keyboard library (KeyboardLibrary)
- Mouse 3D library (Mouse3Dlibrary)
- Windows Mobile Phone library (Windows Mobile Phone)
