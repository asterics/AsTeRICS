# Modes

The ACS can enter several different modes, depending on the status of the connected ARE. The connection status is displayed at the bottom left corner of the ACS. Possible ACS modes are:

*   [Disconnected][1]
*   [Connected][2]
*   [Synchronized][3]
*   [Running][4]
*   [Pause][5]
