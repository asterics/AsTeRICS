

This folder contains subfolders with driver files (.inf files) 
for CIM modules manufactured by IMA 
(see http://www.asterics.eu/index.php?id=32).


For a description of the different CIM modules 
(DigitalIn, DigitalOut, AnalogIn, ZigBee, Wireless DigitalIn etc.) 
please refer to the AsTeRICS User manual, section "hardware".
